## Topics Covered

### ✅ Line Plot

```python
plt.plot()
```

### ✅ Labels

```python
plt.xlabel()
plt.ylabel()
```

### ✅ Subplots

```python
plt.subplot()
```

### ✅ Object Oriented Plotting

```python
fig = plt.figure()
fig.add_axes()
```

### ✅ Figure Size & DPI

```python
figsize
dpi
```

### ✅ Saving Figures

```python
plt.savefig()
```

### ✅ Image Visualization
